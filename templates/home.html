The Evaluation Framework

A holistic scorecard for every agent.

Like a Yelp review with structured dimensions — not just thumbs up or down.

✅

Task Performance

30% weight

Did the agent fully and correctly complete the task? We measure goal achievement, accuracy, output quality, and tool use correctness.

🧠

Reasoning & Autonomy

20% weight

Quality of thinking and decision chains. Logical consistency, handling of ambiguous inputs, and appropriate escalation to humans.

🛡️

Operational Reliability

15% weight

Does it break? Error handling, loop detection, graceful degradation, crash recovery, and retry behavior under edge cases.

👤

User Experience & Trust

15% weight

Tone, clarity, and calibrated confidence. Does the agent communicate uncertainty appropriately? Is it actually helpful to end users?

🔒

Ethics & Safety

10% weight

No harmful outputs, hallucinations, bias, or policy violations. Critical for regulated industries including healthcare, legal, and finance.

⚡

Efficiency

10% weight

Cost per task, unnecessary tool calls, and token waste. Requires trace metadata — null if not provided. Important at production scale.

Each dimension scored 0.0–1.0. Weighted composite = your agent's grade.

FAQ

Common questions.

What trace format do you require?

Any JSON format with a messages array of {"role": "...", "content": "..."} objects. Tool calls and tool results should be included for the most accurate scoring. We handle any framework's output format.

Does it work with all agent frameworks?

Yes. LangChain, LangGraph, CrewAI, AutoGen, raw API calls, and any custom framework. If your agent produces conversation logs, we can evaluate them.

How is this different from Braintrust or LangSmith?

They monitor and log what happened. We deliver a structured third-party reliability assessment with human domain experts — credible to non-technical stakeholders. Think of it as the difference between a server log and an audit report. We produce something your board, investors, and enterprise customers can read and trust.

What do human reviewers actually do?

They receive the full conversation trace, all 6 AI dimension scores, the LLM's reasoning, and any failure details. They read it as a domain expert and submit an independent PASS or FAIL verdict with notes. Their verdict is the final ground truth for contested traces.

What is the agent scorecard?

After each audit, your agent gets a profile page showing its 6-dimension scores across all audits over time. You can make it public as a trust signal for enterprise buyers, investors, and compliance teams.

Is my data confidential?

Yes. All reviewers sign NDAs before receiving any traces. Your trace data is never used for training. We treat your agent conversations as strictly confidential.

Can I start with just a few traces?

Yes — we offer a 5-trace sample evaluation for $500 so you can see the report format and process before committing to a full audit. Email hello@agenteval.com to get started.
